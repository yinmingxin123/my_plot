========================================
    交互式绘图工具 - 使用说明
========================================

【快速开始】

1. 双击 "交互式绘图工具.exe" 启动程序
2. 等待控制台窗口显示 "正在打开浏览器..."
3. 浏览器会自动打开应用界面
4. 开始使用！

【基本操作】

一、加载数据
   - 点击左侧 "上传CSV或Excel文件"
   - 选择你的数据文件
   - 确保第一行是列名

二、创建图表
   1. 填写图表标题
   2. 选择图表类型（折线图/散点图）
   3. 选择 X 轴（横坐标）
   4. 选择 Y1 轴（左侧纵坐标）
   5. 可选：选择 Y2 轴（右侧纵坐标）
   6. 选择数值小数位数
   7. 点击 "生成图表"

三、交互操作
   - 鼠标拖动：选择区域缩放
   - 鼠标悬停：查看所有曲线的值
   - 双击：重置视图
   - 滚轮：在Y轴上滚动可缩放
   - 点击图例：显示/隐藏曲线

【注意事项】

✓ 不需要安装 Python
✓ 不需要安装其他软件
✓ 所有功能都包含在程序中

⚠ 不要单独复制 exe 文件
⚠ 请保持文件夹完整
⚠ 关闭控制台窗口会停止服务

【常见问题】

Q: 浏览器没有自动打开？
A: 查看控制台窗口显示的地址，手动在浏览器中打开

Q: 提示防火墙警告？
A: 点击"允许访问"，程序需要启动本地服务器

Q: 杀毒软件提示风险？
A: 这是误报，添加到信任列表即可

Q: 如何关闭程序？
A: 关闭控制台窗口即可

【系统要求】

- Windows 7/8/10/11（64位）
- 至少 2GB 内存
- 至少 500MB 磁盘空间

【示例数据】

程序包含示例数据文件 "sample_data.csv"
可以用它测试各项功能

【技术支持】

如有问题，请检查：
1. 整个文件夹是否完整
2. 是否被杀毒软件拦截
3. 防火墙是否允许访问
