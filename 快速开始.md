# 🚀 快速开始指南

## 📌 三种使用方式

### 方式 1️⃣：直接运行 Python（开发测试）

适合：开发者、测试功能

```bash
cd E:\python_projects\my_plot
pip install -r requirements.txt
streamlit run app.py
```

### 方式 2️⃣：使用启动器（方便启动）

适合：本地使用，不想打包

**Windows:**
- 双击 `测试启动器.bat`

**或者命令行:**
```bash
python launcher.py
```

### 方式 3️⃣：打包成 exe（分发给他人）

适合：分享给没有 Python 环境的用户

**Windows 一键打包:**
1. 双击 `一键打包.bat`
2. 等待打包完成
3. 进入 `dist/交互式绘图工具/`
4. 双击 `交互式绘图工具.exe` 测试
5. 压缩整个文件夹发送给他人

**命令行打包:**
```bash
pip install -r requirements_build.txt
python build_exe.py
```

## 📊 基本使用流程

### 1. 启动应用
- 选择上面任一方式启动
- 浏览器自动打开（或手动访问显示的地址）

### 2. 上传数据
- 左侧边栏点击"上传CSV或Excel文件"
- 选择数据文件（第一行必须是列名）

### 3. 创建图表
- 填写图表标题
- 选择图表类型（折线图/散点图）
- 选择 X 轴
- 选择 Y1 轴（一个或多个列）
- 可选：选择 Y2 轴（右侧独立坐标）
- 选择小数位数（0-6位）
- 点击"生成图表"

### 4. 交互操作
- **鼠标悬停**：看到纵向虚线和所有曲线的值
- **拖动选择**：框选区域进行缩放
- **滚轮**：在Y轴上滚动可缩放
- **双击**：重置视图
- **点击图例**：显示/隐藏曲线

## 🎁 分发给他人

### 打包步骤
1. 双击 `一键打包.bat`（或运行 `python build_exe.py`）
2. 等待打包完成
3. 测试 `dist/交互式绘图工具/交互式绘图工具.exe`

### 分发步骤
1. 将 `dist/交互式绘图工具/` **整个文件夹**压缩成 zip
2. 通过邮件、网盘等发送
3. 告诉对方：
   - 解压文件
   - 双击 `交互式绘图工具.exe`
   - 允许防火墙访问
   - 浏览器会自动打开

### ⚠️ 重要提示
- **必须保持整个文件夹完整**
- 不要单独复制 exe 文件
- exe 需要 `_internal` 文件夹中的依赖

## ❓ 常见问题

### Q1: 浏览器没有自动打开？
**A:** 查看控制台显示的地址，手动在浏览器中打开，通常是 `http://localhost:端口号`

### Q2: 防火墙提示？
**A:** 点击"允许访问"，应用需要启动本地服务器

### Q3: 杀毒软件报毒？
**A:** 这是误报（PyInstaller 打包的程序常见问题），添加到信任列表

### Q4: exe 无法运行？
**A:** 
- 确保整个文件夹完整
- 以管理员身份运行
- 检查是否被杀毒软件拦截

### Q5: 打包失败？
**A:**
- 确保已安装 `requirements_build.txt` 中的依赖
- 删除 `build`、`dist` 文件夹后重试
- 升级 PyInstaller：`pip install --upgrade pyinstaller`
